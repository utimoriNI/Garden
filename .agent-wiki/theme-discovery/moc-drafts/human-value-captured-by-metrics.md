# 指標に回収される人間の価値

Created: 2026-04-11
Status: draft
Source theme proposal: [[.agent-wiki/theme-discovery/themes/life-society-proposals-01]]

## Scope

この MOC 草案は、個人の価値や能力や努力が、その人自身の厚みではなく、
社会的に見えやすい指標や評価可能性に回収されてしまう現象を束ねる。

中心にある問いは次の通り。

- 人間はいつ「その人自身」ではなく「指標の束」として扱われるのか
- 能力主義や成果主義は、何を可視化し、何を切り捨てるのか
- その評価装置は、どのように自己評価の形式まで変えてしまうのか

## 1. 個人は指標と結びつけられて価値づけられる

- [[300_Input/Reading Notes/ふくさん_年収の話しかしないビジネスアカウント，生きてるだけでえらいなどの文言，これらの根底にあるのは「個人はある指標と紐づけられて評価される」という格の価値観]]
- [[300_Input/Reading Notes/今や外見の良さも「努力の証」として能力主義的に評価されている]]

メモ:
ここでは年収、外見、肩書きのような見えやすいものが、そのまま人格的価値の代理変数として扱われている。

## 2. 成果主義は「見える努力」だけを残す

- [[300_Input/Reading Notes/成果主義は頑張ったアピールをしないと切られるルール]]
- [[300_Input/Reading Notes/体調最悪でも2日寝てなくても友達に裏切られても女にフラれてもその中で 歯くいしばってひねり出した仕事がお前の実力のすべてだ。クソみたいな日にいいもんつくるのがプロだ。]]

メモ:
ここでは成果主義が単なる評価制度ではなく、「どう努力すべきか」「どの苦しみが正当と見なされるか」まで規定している。

## 3. 成功と能力の評価は自己像にも侵入する

- [[300_Input/Reading Notes/インポスター症候群]]
- [[300_Input/Reading Notes/人付き合いって大事かなと思ったら読みたい3冊]]

メモ:
評価は他人からのラベリングにとどまらず、自分自身が自分をどう見積もるかにも入り込む。成功は能力だけでなく、承認回路や認知バイアスにも支えられる。

## ここから作れそうな Knowledge Note

- 指標は能力そのものではなく、評価されやすさを増幅する
- 能力主義は努力を称賛するのではなく、可視化可能な努力だけを正当化する
- 自己評価の歪みは、社会的評価装置の内面化として理解できる
- 成果主義は成果を測る制度である以上に、人格の読み方を変える制度でもある
