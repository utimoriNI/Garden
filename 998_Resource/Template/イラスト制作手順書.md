## 2. 工程別手順

### 2.1 目的・資料

**この工程の目的**

- 完成後に見せたい印象を決める。
- 迷ったときに戻る判断基準を作る。

**記録すること**

| 項目        | 内容  |
| --------- | --- |
| テーマ       |     |
| 見せたい感情    |     |
| 視線を集めたい場所 |     |
| 主役の魅力     |     |
| 色の方向性     |     |
| 光源        |     |
| 構図の候補     |     |

**終了条件**

- [ ] 何を描くか一文で言える
- [ ] 主役と視線誘導の場所が決まっている
- [ ] 参考資料が最低3つある

### 2.2 骨格ラフ

**この工程の目的**

- ポーズ、重心、画面内の配置を決める。
- 細部ではなく、絵の大きな設計を確認する。

**手順**

- [ ] 太めのブラシで人物や物体のアタリを取る
- [ ] 重心、顔の向き、肩、腰、手足の位置を確認する
- [ ] 画面の余白と主役の大きさを確認する
- [ ] 複数案を作る場合は A / B / C で分ける

**終了条件**

- [ ] ポーズの意図が分かる
- [ ] 画面内の主役の位置が決まっている
- [ ] 次の大ラフで詰める案を選べる

**ログ**

- 迷ったこと:
- 採用した理由:
- 次回変えたいこと:

### 2.3 大ラフ

**この工程の目的**

- 骨格ラフを人間らしい形、または完成形に近いシルエットへ寄せる。
- 構図、カメラアングル、シチュエーションを決める。

**手順**

- [ ] 骨格ラフの不透明度を下げる
- [ ] 新規レイヤーに大ラフを描く
- [ ] 顔、髪、服、小物、背景の大まかな位置を入れる
- [ ] 表情とシルエットが伝わるか確認する

**終了条件**

- [ ] 何をしている絵か分かる
- [ ] 顔・手・髪・服の大まかな配置が決まっている
- [ ] 背景や小物の有無が決まっている

### 2.4 詳細ラフ

**この工程の目的**

- 線画に進んでもズレにくい状態まで、表情・服・手・髪の流れを詰める。

**手順**

- [ ] 大ラフの不透明度を下げる
- [ ] 線画で迷いそうな部分を先に決める
- [ ] 顔、目、口、手、髪、服の重なりを整理する
- [ ] 背景と人物の接点を確認する

**終了条件**

- [ ] 線画で拾う線がほぼ分かる
- [ ] 手、顔、髪などの難所が曖昧なまま残っていない
- [ ] ここで描いた良い線を線画で残す方針が決まっている

### 2.5 陰影ラフ / カラーラフ

**この工程の目的**

- 線画後に完成イメージがズレないように、明暗と色の設計を先に置く。

**手順**

- [ ] ラフの下に新規レイヤーを作る
- [ ] 大きな明暗を置く
- [ ] ベースカラーを仮置きする
- [ ] キャラクターと背景を別レイヤーに分ける
- [ ] 光源と影の方向をメモする

**終了条件**

- [ ] 完成時の雰囲気が見える
- [ ] 明るい場所、暗い場所、視線を集める場所が決まっている
- [ ] 後から色調整しやすいレイヤー分けになっている

### 2.6 線画

**この工程の目的**

- ラフの良い線を残しつつ、塗りやすく修正しやすい線画にする。

**手順**

- [ ] ラフ / 下描きの不透明度を下げる
- [ ] 必要ならレイヤーカラーを使って下描きを見やすくする
- [ ] 線画フォルダーを作る
- [ ] パーツ別にレイヤーを分ける
- [ ] 髪や服など勢いが必要な線は、あとで消しゴム修正する前提で描く
- [ ] 背景や直線部分は別レイヤーにする

**レイヤー例**

- 線画 / 顔
- 線画 / 髪
- 線画 / 体
- 線画 / 服
- 線画 / 小物
- 線画 / 背景

**終了条件**

- [ ] 塗りつぶしに困る大きな線の隙間がない
- [ ] 線の強弱が主役の見せたい場所に合っている
- [ ] 不要なラフ線が非表示になっている

### 2.7 下塗り

**この工程の目的**

- パーツごとの色分けを作り、以降の塗りをはみ出さず進められる状態にする。

**手順**

- [ ] 線画の下に下塗りフォルダーを作る
- [ ] 背景色を置く
- [ ] 肌、髪、服、小物、背景でレイヤーを分ける
- [ ] 塗り残しを確認する
- [ ] 必要ならラフ色を複製・クリッピングして色の方向性を残す

**終了条件**

- [ ] パーツごとに選択しやすい
- [ ] 塗り残しがない
- [ ] ラフ時点の色の印象が失われていない

### 2.8 パーツ塗り

**この工程の目的**

- 主役の魅力が出る順番で、立体感と質感を作る。

**順番**

- [ ] 目
- [ ] 肌
- [ ] 髪
- [ ] 服
- [ ] 小物
- [ ] 背景

**各パーツで記録すること**

| パーツ | 使ったブラシ | レイヤーモード | 影色 | ハイライト | 注意点 |
| --- | --- | --- | --- | --- | --- |
| 目 |  |  |  |  |  |
| 肌 |  |  |  |  |  |
| 髪 |  |  |  |  |  |
| 服 |  |  |  |  |  |
| 背景 |  |  |  |  |  |

**終了条件**

- [ ] 主役の顔まわりに視線が行く
- [ ] 明暗の幅が足りている
- [ ] パーツごとの質感差がある

### 2.9 背景 / 前景 / 効果

**この工程の目的**

- 画面の奥行き、空気感、視線誘導を足す。

**手順**

- [ ] 背景の明暗がキャラクターを邪魔していないか確認する
- [ ] 前景要素を入れる場合は最上部にレイヤーを作る
- [ ] ぼかし、スクリーン、オーバーレイなどの効果を試す
- [ ] 効果レイヤーの不透明度を下げてやりすぎを防ぐ

**終了条件**

- [ ] 主役が背景に埋もれていない
- [ ] 手前、主役、奥の距離感がある
- [ ] 効果が絵のテーマに合っている

### 2.10 仕上げ / 色調整

**この工程の目的**

- 全体の色、明暗、視線誘導を統一して完成度を上げる。

**手順**

- [ ] 全体を縮小表示して確認する
- [ ] 左右反転して崩れを確認する
- [ ] 顔、手、主役の輪郭を確認する
- [ ] 線画色を部分的になじませる
- [ ] オーバーレイや色調補正で全体の色を統一する
- [ ] 彩度が高すぎる場所、暗すぎる場所を調整する
- [ ] 書き出しサイズで見て破綻がないか確認する

**終了条件**

- [ ] サムネイルでも主役が分かる
- [ ] 視線が最初に行く場所が意図通り
- [ ] 色味がバラバラに見えない
- [ ] 完成として出せる
