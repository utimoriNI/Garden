# イラスト制作ログ

## 使い方

このノートは、制作中にきれいな文章を書くためのものではない。
制作中は `1. クイックログ` と `2. 今のタスク` だけを使い、完成後または休憩時に `4. 整理後の制作手順` へ転記する。

制作中のメモは、短く、雑でよい。

- `- [ ]` やること
- `- [x]` 終わったこと
- `- ?` 迷ったこと
- `- !` 気づき、次回も使いたいこと
- `- fix:` 修正したいこと
- `- next:` 次にやること
