>> ## 色選びについて
>> - **ハイキー**
>> 	- 彩度5~30くらい
>> 		- 中明度で彩度が高くなる
>> 		- 10, 15, 25くらいで暗くしていく
>> 	- 明度25~80くらい
>> 		- 均等に下げていく感じで良い
>> 		- ベース: 60~70
>> 		- コントラストを高めたいときは40くらいを使ってみてもよさそう
>> 			- 彩度50と同じでピンポイントのピック
>> - **ローキー**
>> 	- 基本的にはハイキーの場合と使う範囲は一緒
>> 	- 配分を変えることでローキーにできる
>> 	- めっちゃ暗い絵の時は明度を50~60を多く使って彩度を上げる(40~55くらい)

### オーバーレイの色変化調査
#### オーバーレイの仕組み
- 乗算とスクリーンを組み合わせたレイヤーモード
- 基本色（塗られる側）のRGB値が、128未満は乗算、128以上はスクリーンが適用される
- 基本色 < 128 の場合
	- **結果色 = 基本色 * 合成色 / 255 * 2**  
- 基本色 ≧ 128 の場合  
	- **結果色 = 2 * (基本色 + 合成色 – 基本色 * 合成色 / 255) – 255**
