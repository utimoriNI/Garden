### コード進行
#### イントロ
Dadd9(4) -> E(5) -> F#m7(11)(6) -> C#m7(3) -> E(5) -> F#m7(11)(6)
- 4-5-6進行とやらが王道らしい
- 4-5-6って上がっていく力強さがある
	- 4-5-6-1とメジャートニックに着地すると明るい印象
	- 4-5-6-3とマイナートニックに着地すると暗い印象

#### Aメロ
- とりあえずセブンスか9を使ってる
- ピアノはコードの一部の音が省略されて弾かれてる なぜ
	- 省略されてた方が抜けが良くてすっきり聞こえる気がする
	- DM7(9)でF#が省略されてる
		- 3度の音を省略することでメジャーかマイナーかをあいまいにして浮遊感を出せるらしい
	- Eadd9でG#が省略されてる
		- 同じく3度が省略されてる
- でしょうか～まで
	- DM7(9) -> Eadd9 -> F#m7 -> C#m7 ->  DM7(9) ->Eadd9 -> F#m7 ->GM7
	- 4-5-6-3-4-5-6-7
		- 4-5-6-3から4-5-6-7って感じ

#### Bメロ
- Bm7 -> C#m7 -> DM7 -> C#m7 -> Bm7 ->C#m7 -> DM7 ->C#7(ノンダイアトニック)
- 2-3-4-3-2-3-4-3(ノンダイアトニック)
