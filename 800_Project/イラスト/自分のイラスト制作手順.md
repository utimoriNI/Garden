# 自分のイラスト制作手順
## 使い方
このノートは、作品ごとの制作ログから抽出した「自分が絵を描くときの説明書」。

- 作品ごとの制作ログに、制作中メモ、整理後の制作手順、迷い、振り返りを残す
- 汎用化できる学びだけ、このノートの `2. 標準制作手順` に反映する
- 迷ったことはすぐ標準化せず、`3. 採用判断中` に置く
- 作品固有の技法は標準手順に混ぜず、`4. Tips集` に置く
- 制作ノートを増やすたび、このノートを少しずつ更新する

## 2. 標準制作手順
### 2.1 目的・資料
**目的**
- 何を描くかを一文で決める
- 完成目標と今回の実験を分けて書く
- 今回はやらないことを決めて、制作中の迷いを減らす

**手順**
1. 作品の主役を決める
2. 見せたい感情、魅力、視線を集めたい場所を書く
3. 参考資料を集める
4. 色の方向性、光源、構図候補を仮で決める

**チェック**
- [ ] 何を描くか一文で言える
- [ ] 主役と視線誘導の場所が決まっている
- [ ] 参考資料がある

### 2.2 ラフ
**目的**
- 完成時の構図、明暗、顔まわりの見え方を早い段階で確認する

**手順**
1. 骨格、重心、顔の向きを決める
2. 大ラフで構図とカメラアングルを決める
3. 髪や服は最初に大きなシルエットで取る
4. グレーで明暗を分ける
5. カラーラフに進む前に、主役の顔まわりに視線が行くか確認する
	1. 必要に応じて明暗構成やシルエットを調整する

**チェック**
- [ ] 骨格、重心、顔の向きが決まっている
- [ ] 大ラフで構図、カメラアングル、シチュエーションが分かる
- [ ] 陰影ラフまたはカラーラフで完成時の雰囲気が見える

### 2.3 カラーラフ
**目的**
- ベース色、影色、光源色の関係を早めに決める

**手順**
1. 明部のベース色を置く
	1. 肌など基準にしやすい色から決める
2. 影は乗算レイヤーで置き、パーツごとに色を調整する
	1. 乗算レイヤーの色はベースの色の明度を落とした色をまず置く
	2. 必要なら色調補正で影色を調整する
3. ベース、影、反射光、ハイライト、オクルージョンシャドウのレイヤー構成で確認する
	1. ディティールは描き込まない
	2. MB Cube Brushがちょうどいいディティール

**チェック**
- [ ] 主光源の色を意識して色を選んでいる
- [ ] 影色がパーツごとに破綻していない
- [ ] サムネイルで明暗構成が読める

### 2.4 線画
**目的**
- 塗りで形を作るための最低限のガイドを作る

**手順**
1. 服、髪の毛束など前後関係を分かりやすくしたいところは線を描く
2. 塗りで形を作れる部分は、線を描き込みすぎない
3. 線画の時点で細部に入りすぎた場合は、下描き程度で止める選択肢を検討する

**チェック**
- [ ] 塗りで作る部分と線で作る部分を分けている
- [ ] 線画に時間をかけすぎていない

### 2.5 下塗り
**目的**
- パーツごとの塗り分けと、後工程で調整しやすいレイヤー構成を作る

**手順**
1. パーツごとにベース色を分ける
2. 髪、肌、服、小物、背景を別々に調整できる状態にする
3. ベースの色は影の色を置く

**チェック**
- [ ] パーツごとに選択、調整しやすい
- [ ] カラーラフの印象が残っている

### 2.6 パーツ塗り
**目的**
- 主役の魅力が出る順番でディテールを足す

**手順**
1. 顔まわりを優先して塗る
2. 目、肌、髪の順に印象を固める
3. 服や小物は主役を邪魔しない範囲で情報量を調整する
4. 髪は大きな束を先に作り、あとから細い毛やハイライトを足す

**チェック**
- [ ] 主役の顔まわりに視線が行く
- [ ] 明暗の幅が足りている
- [ ] パーツ同士の情報量に差がありすぎない

### 2.7 背景・効果
**目的**
- 主役を見やすくし、画面全体の距離感と雰囲気を作る

**手順**
1. 主役が背景に埋もれていないか確認する
2. 手前、主役、奥の距離感を作る
3. 効果は絵のテーマに合うものだけ足す

**チェック**
- [ ] 主役が背景に埋もれていない
- [ ] 手前、主役、奥の距離感がある
- [ ] 効果が絵のテーマに合っている

### 2.8 仕上げ
**目的**
- 視線誘導、色味、サムネイルでの読みやすさを整える

**手順**
1. 顔、手、主役の輪郭を確認する
2. 色味がバラバラに見えないよう調整する
3. サムネイルで主役が分かるか確認する
4. 最後に一番見せたい場所のコントラストを確認する

**チェック**
- [ ] 顔、手、主役の輪郭を確認した
- [ ] 色味がバラバラに見えない
- [ ] サムネイルでも主役が分かる

## 3. 採用判断中
迷ったことは、制作直後にここへ置く。2作品以上で有効だったら標準手順に入れる。

| 候補                  | 出た作品             | 入れる工程      | 判断   | 判断理由                          | 次に見ること          |
| ------------------- | ---------------- | ---------- | ---- | ----------------------------- | --------------- |
| 模写でもカラーラフを作る        | [[bbul_horn_模写]] | ラフ / カラーラフ | 保留   | 模写なら省略できるが、カラーラフ練習としては価値がありそう | 次の模写でも時間と効果を見る  |
| 線画を下描き程度に止め、塗りで形を作る | [[bbul_horn_模写]] | 線画         | 保留   | 線画の負担を減らせそうだが、完成度への影響を確認したい   | 顔まわりと髪で破綻しないか見る |
| 肌色から全体の色を決める        | [[bbul_horn_模写]] | カラーラフ      | 採用候補 | 最初の1色を決めやすく、全体トーンの基準にしやすい     | 他の光源でも使えるか見る    |
| 境界効果で線画を描く          |                  |            |      |                               |                 |

## 4. Tips集
標準手順には入れないが、似た題材や質感で再利用できる処理。

| Tips | 関連作品 | 使う場面 | 手順 | 注意点 |
| --- | --- | --- | --- | --- |
| MB CubeBrushで影を削る | [[bbul_horn_模写]] | カラーラフの影整理 | 乗算で置いた影を、MB CubeBrushで削って形を作る | ディテールより先に大きい影の形を作る |
| 3Dモデルで顔の向きを確認する | [[bbul_horn_模写]] | ラフ | 顔の角度に迷うとき、3Dモデルを補助資料にする | モデルに合わせすぎず、最終的な絵柄に合わせて調整する |
