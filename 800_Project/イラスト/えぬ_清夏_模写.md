# イラスト制作ログ
## 使い方
制作中は `1. クイックログ` と `2. 今のタスク` だけを使い、完成後または休憩時に `4. 整理後の制作手順` へ転記する。

制作中のメモは、短く、雑でよい。

- `- [ ]` やること
- `- [x]` 終わったこと
- `- ?` 迷ったこと
- `- !` 気づき、次回も使いたいこと
- `- fix:` 修正したいこと
- `- next:` 次にやること

## 1. クイックログ
制作中はここにだけ書く。
時刻、工程名、整った文章はなくてよい。

### ログ
- 影をベースにするか光をベースにするか未だに迷う
	- 反射光レイヤーの場所
	- 1枚でやると、全体のグラデーションや影部分のみの指定などができない
	- モデル的には影ベースが自然
		- 影ベースだとフォームシャドウが描きにくい
	- 光ベースがいいかもな

### 4.6 パーツ塗り

- カラーラフで作った影の形を元に明暗を分ける
