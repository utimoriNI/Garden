# イラスト制作ログ

## 使い方

制作中は `1. クイックログ` と `2. 今のタスク` だけを使い、完成後または休憩時に `4. 整理後の制作手順` へ転記する。

制作中のメモは、短く、雑でよい。

- `- [ ]` やること
- `- [x]` 終わったこと
- `- ?` 迷ったこと
- `- !` 気づき、次回も使いたいこと
- `- fix:` 修正したいこと
- `- next:` 次にやること

### 4.4 カラーラフ

**やったこと**
- 影の色でベースを塗ってその後に光を当てる方法をやめてみた
	- 1色目を選ぶのが難しいが、それさえ決まれば分かりやすいかも
		- 肌の色を最初に決めると、選びやすいし全体のトーンも決めやすい？
	- 明暗構成をちゃんとやっていれば、ここから影の色を選ぶ時も破綻しにくいかも
	- 主光源の色を考えて色を決めればいいから楽
- 
- ベースの色を影の暗さで塗ってるけど、普通にベースの色でいいかも？
	- カラーラフ制作の時は普通の色で色置いて、乗算で影を置けるといいかも
		- 乗算レイヤーを1色で影入れするのは無理でした
		- パーツごとに色の調整が必要
		- つまり影色をベースで入れるか、明部の色をベースで入れるかの違いしかない
	- [Site Unreachable](https://x.com/mochi_upamo/status/2062711517070717058/photo/1)
	- ![image](https://garden-storage-228689237525-us-east-2-an.s3.us-east-2.amazonaws.com/ef8a9280227fdb106c427b593ee3ea47.png)
	- ベースの色を明部の色で塗って乗算レイヤーで影を置くのが楽
	- 乗算レイヤーで塗る色はベースの色の明度を下げた色
		- そこから色調補正で調整する
	- MB CubeBrushでトントンして影を削るといい
		- ディティールを気にしないで影を作れる
	- カラーラフのレイヤー構成
		- ハイライト
		- 反射光
		- 影(乗算)
		- ベース
