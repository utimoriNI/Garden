# [彼氏がおもんなさすぎる](https://web.archive.org/web/20241002044600/https://anond.hatelabo.jp/20241002111546)

### [■](https://web.archive.org/20241002111546)[彼氏](https://web.archive.org/keyword/%E5%BD%BC%E6%B0%8F)がおもんなさすぎる

[念の為](https://web.archive.org/keyword/%E5%BF%B5%E3%81%AE%E7%82%BA)、彼の顔や体型、[学歴](https://web.archive.org/keyword/%E5%AD%A6%E6%AD%B4)や[社会](https://web.archive.org/keyword/%E7%A4%BE%E4%BC%9A)性[地位](https://web.archive.org/keyword/%E5%9C%B0%E4%BD%8D)には[言及](https://web.archive.org/keyword/%E8%A8%80%E5%8F%8A)[しま](https://web.archive.org/keyword/%E3%81%97%E3%81%BE)せん。[関係](https://web.archive.org/keyword/%E9%96%A2%E4%BF%82)ないので。
彼とはマ[チア](https://web.archive.org/keyword/%E3%83%81%E3%82%A2)プで[出会い](https://web.archive.org/keyword/%E5%87%BA%E4%BC%9A%E3%81%84)ました。
[最初](https://web.archive.org/keyword/%E6%9C%80%E5%88%9D)の[デート](https://web.archive.org/keyword/%E3%83%87%E3%83%BC%E3%83%88)の[とき](https://web.archive.org/keyword/%E3%81%A8%E3%81%8D)[から](https://web.archive.org/keyword/%E3%81%8B%E3%82%89)、会話があ[まり](https://web.archive.org/keyword/%E3%81%BE%E3%82%8A)[面白](https://web.archive.org/keyword/%E9%9D%A2%E7%99%BD)くないな...と思ってたんですが
5回目の[デート](https://web.archive.org/keyword/%E3%83%87%E3%83%BC%E3%83%88)（5回目[！？](https://web.archive.org/keyword/%EF%BC%81%EF%BC%9F)）で彼[から](https://web.archive.org/keyword/%E3%81%8B%E3%82%89)[告白](https://web.archive.org/keyword/%E5%91%8A%E7%99%BD)されてお付き合いが始[まり](https://web.archive.org/keyword/%E3%81%BE%E3%82%8A)ましたが
会う度に「こいつおもんな～～～」って思う。
#### ■なにがおもんないのか

[人間](https://web.archive.org/keyword/%E4%BA%BA%E9%96%93)誰しも、[普通に](https://web.archive.org/keyword/%E6%99%AE%E9%80%9A%E3%81%AB)生きてたら[鉄板](https://web.archive.org/keyword/%E9%89%84%E6%9D%BF)の話が有ると思い[ます](https://web.archive.org/keyword/%E3%81%BE%E3%81%99)。
[定義](https://web.archive.org/keyword/%E5%AE%9A%E7%BE%A9)は[ウケる](https://web.archive.org/keyword/%E3%82%A6%E3%82%B1%E3%82%8B)ウケないは別として、[絶対](https://web.archive.org/keyword/%E7%B5%B6%E5%AF%BE)に関心を引くような話です。
（私の[場合](https://web.archive.org/keyword/%E5%A0%B4%E5%90%88)は、マ[チア](https://web.archive.org/keyword/%E3%83%81%E3%82%A2)プで[出会い](https://web.archive.org/keyword/%E5%87%BA%E4%BC%9A%E3%81%84)2年半付き合った[モラハラ](https://web.archive.org/keyword/%E3%83%A2%E3%83%A9%E3%83%8F%E3%83%A9)男が[出会](https://web.archive.org/keyword/%E5%87%BA%E4%BC%9A)った[とき](https://web.archive.org/keyword/%E3%81%A8%E3%81%8D)[から](https://web.archive.org/keyword/%E3%81%8B%E3%82%89)既婚者子持ちだった）
（ずっと母方のお婆[ちゃん](https://web.archive.org/keyword/%E3%81%A1%E3%82%83%E3%82%93)だと思ってた[女性](https://web.archive.org/keyword/%E5%A5%B3%E6%80%A7)が実は[赤の他人](https://web.archive.org/keyword/%E8%B5%A4%E3%81%AE%E4%BB%96%E4%BA%BA)だった）
（[ハリウッド](https://web.archive.org/keyword/%E3%83%8F%E3%83%AA%E3%82%A6%E3%83%83%E3%83%89)[スター](https://web.archive.org/keyword/%E3%82%B9%E3%82%BF%E3%83%BC)との[写真](https://web.archive.org/keyword/%E5%86%99%E7%9C%9F)[撮影](https://web.archive.org/keyword/%E6%92%AE%E5%BD%B1)で、[プリンター](https://web.archive.org/keyword/%E3%83%97%E3%83%AA%E3%83%B3%E3%82%BF%E3%83%BC)の[エラー](https://web.archive.org/keyword/%E3%82%A8%E3%83%A9%E3%83%BC)で[写真](https://web.archive.org/keyword/%E5%86%99%E7%9C%9F)が出ず、5万円を[無駄](https://web.archive.org/keyword/%E7%84%A1%E9%A7%84)にした）（これは今でも本当に悲しい）
彼はこういった[エピソード](https://web.archive.org/keyword/%E3%82%A8%E3%83%94%E3%82%BD%E3%83%BC%E3%83%89)を一つも持っていません。
[会社](https://web.archive.org/keyword/%E4%BC%9A%E7%A4%BE)でも、[上司](https://web.archive.org/keyword/%E4%B8%8A%E5%8F%B8)や後輩がみんないい人だそうで、[やらかし](https://web.archive.org/keyword/%E3%82%84%E3%82%89%E3%81%8B%E3%81%97)や[トラブル](https://web.archive.org/keyword/%E3%83%88%E3%83%A9%E3%83%96%E3%83%AB)も穏便に済ませてきたらしく
飲みの席での[所謂](https://web.archive.org/keyword/%E6%89%80%E8%AC%82)「[武勇伝](https://web.archive.org/keyword/%E6%AD%A6%E5%8B%87%E4%BC%9D)」という[もの](https://web.archive.org/keyword/%E3%82%82%E3%81%AE)ありません。
彼が[主人公](https://web.archive.org/keyword/%E4%B8%BB%E4%BA%BA%E5%85%AC)の[エピソード](https://web.archive.org/keyword/%E3%82%A8%E3%83%94%E3%82%BD%E3%83%BC%E3%83%89)トークを聞[いたこ](https://web.archive.org/keyword/%E3%81%84%E3%81%9F%E3%81%93)とがないです。
彼が[友達](https://web.archive.org/keyword/%E5%8F%8B%E9%81%94)と[旅行](https://web.archive.org/keyword/%E6%97%85%E8%A1%8C)に行き、[感想](https://web.archive.org/keyword/%E6%84%9F%E6%83%B3)を聞けば「◯◯に行って、◯◯見て～、◯◯食べて、寝た」
私が「◯◯には行った？」と聞けば「行かなかった」で終了です。
話を全く広げない。

#### ■いやそれお前がおもんないだけ[ちゃう](https://web.archive.org/keyword/%E3%81%A1%E3%82%83%E3%81%86)ん？

一理あると思い[ます](https://web.archive.org/keyword/%E3%81%BE%E3%81%99)。
ですが、私は少なくとも[旅行](https://web.archive.org/keyword/%E6%97%85%E8%A1%8C)の[エピソード](https://web.archive.org/keyword/%E3%82%A8%E3%83%94%E3%82%BD%E3%83%BC%E3%83%89)があれば
「◯◯に行ったんだけど、途中で雨が降ってきて～でも強行[突破](https://web.archive.org/keyword/%E7%AA%81%E7%A0%B4)で◯◯見よう！ってなって～
[日傘](https://web.archive.org/keyword/%E6%97%A5%E5%82%98)[しか](https://web.archive.org/keyword/%E3%81%97%E3%81%8B)持ってなかっ[たか](https://web.archive.org/keyword/%E3%81%9F%E3%81%8B)ら[日傘](https://web.archive.org/keyword/%E6%97%A5%E5%82%98)びちょびちょになった笑
雨やばかっ[たか](https://web.archive.org/keyword/%E3%81%9F%E3%81%8B)らさすがに[タクシー](https://web.archive.org/keyword/%E3%82%BF%E3%82%AF%E3%82%B7%E3%83%BC)で帰った[笑笑](https://web.archive.org/keyword/%E7%AC%91%E7%AC%91)
それでそのあと◯◯食べて～、あ、[写真](https://web.archive.org/keyword/%E5%86%99%E7%9C%9F)これ！◯◯が入っててそれが美味[しか](https://web.archive.org/keyword/%E3%81%97%E3%81%8B)った！」ぐらいは話[しま](https://web.archive.org/keyword/%E3%81%97%E3%81%BE)す。
私は「[自分](https://web.archive.org/keyword/%E8%87%AA%E5%88%86)の話はおもんない」を前提として、精一杯楽しんで貰えるように話し方や[段落](https://web.archive.org/keyword/%E6%AE%B5%E8%90%BD)には気をつけて喋ってい[ます](https://web.archive.org/keyword/%E3%81%BE%E3%81%99)。
彼には聞き[相手](https://web.archive.org/keyword/%E7%9B%B8%E6%89%8B)を楽[しま](https://web.archive.org/keyword/%E3%81%97%E3%81%BE)せようという心意気を一切感じません。
ChatGPTと話してるみたい。

#### ■じゃあ別れれば？

彼とは[趣味](https://web.archive.org/keyword/%E8%B6%A3%E5%91%B3)が一緒で、その同じ[趣味](https://web.archive.org/keyword/%E8%B6%A3%E5%91%B3)をしている[とき](https://web.archive.org/keyword/%E3%81%A8%E3%81%8D)は[楽しい](https://web.archive.org/keyword/%E6%A5%BD%E3%81%97%E3%81%84)ので別れません。

#### ■結局何が言いたいのか

仲がいい[相手](https://web.archive.org/keyword/%E7%9B%B8%E6%89%8B)でも、最小限[自分](https://web.archive.org/keyword/%E8%87%AA%E5%88%86)で[エピソード](https://web.archive.org/keyword/%E3%82%A8%E3%83%94%E3%82%BD%E3%83%BC%E3%83%89)を掘り下げる[努力](https://web.archive.org/keyword/%E5%8A%AA%E5%8A%9B)をして欲しいです。
これは[転職](https://web.archive.org/keyword/%E8%BB%A2%E8%81%B7)や[就職](https://web.archive.org/keyword/%E5%B0%B1%E8%81%B7)時に「[コミュニケーション能力](https://web.archive.org/keyword/%E3%82%B3%E3%83%9F%E3%83%A5%E3%83%8B%E3%82%B1%E3%83%BC%E3%82%B7%E3%83%A7%E3%83%B3%E8%83%BD%E5%8A%9B)」として[PR](https://web.archive.org/keyword/PR)出来る部分です。
笑わせろ！というわけではなく、[聞き手](https://web.archive.org/keyword/%E8%81%9E%E3%81%8D%E6%89%8B)は細[かい](https://web.archive.org/keyword/%E3%81%8B%E3%81%84)部分まで[言及](https://web.archive.org/keyword/%E8%A8%80%E5%8F%8A)してくれた方が会話の糸口を見つけ[やす](https://web.archive.org/keyword/%E3%82%84%E3%81%99)いです。
昨日なにがあった？という[エピソード](https://web.archive.org/keyword/%E3%82%A8%E3%83%94%E3%82%BD%E3%83%BC%E3%83%89)を最大限省略すれば「[仕事](https://web.archive.org/keyword/%E4%BB%95%E4%BA%8B)して、飯食って、寝た」で終わり[ます](https://web.archive.org/keyword/%E3%81%BE%E3%81%99)。
最小限掘り下げて「◯時に起きて～[めざまし](https://web.archive.org/keyword/%E3%82%81%E3%81%96%E3%81%BE%E3%81%97)見ながら準備して～」と話してくれれば
「朝は私[ZIP](https://web.archive.org/keyword/ZIP)派だな～」という糸口に繋がり[ます](https://web.archive.org/keyword/%E3%81%BE%E3%81%99)。
まじで全[人類](https://web.archive.org/keyword/%E4%BA%BA%E9%A1%9E)、[努力](https://web.archive.org/keyword/%E5%8A%AA%E5%8A%9B)してくれ。
