# [面白い数のパラドックス - Wikipedia](https://ja.wikipedia.org/wiki/%E9%9D%A2%E7%99%BD%E3%81%84%E6%95%B0%E3%81%AE%E3%83%91%E3%83%A9%E3%83%89%E3%83%83%E3%82%AF%E3%82%B9)

出典: フリー百科事典『ウィキペディア（Wikipedia）』

**面白い数のパラドックス**（おもしろいかずのパラドックス）は、数学ジョークの一つである。すべての[自然数](https://ja.wikipedia.org/wiki/%E8%87%AA%E7%84%B6%E6%95%B0 "自然数")を「面白い」「面白くない」として分類しようとする試みから生じる。 パラドックスは、「すべての[自然数](https://ja.wikipedia.org/wiki/%E8%87%AA%E7%84%B6%E6%95%B0 "自然数")は面白い」と述べている。 証明は[背理法](https://ja.wikipedia.org/wiki/%E8%83%8C%E7%90%86%E6%B3%95 "背理法")によるものである。面白くない自然数の空でない集合が存在する場合、最小の面白くない数が存在するが、「最小の面白くない数」はそれ自体が面白い性質であるので、矛盾が生じる。

数学者の[ゴッドフレイ・ハロルド・ハーディ](https://ja.wikipedia.org/wiki/%E3%82%B4%E3%83%83%E3%83%89%E3%83%95%E3%83%AC%E3%82%A4%E3%83%BB%E3%83%8F%E3%83%AD%E3%83%AB%E3%83%89%E3%83%BB%E3%83%8F%E3%83%BC%E3%83%87%E3%82%A3 "ゴッドフレイ・ハロルド・ハーディ")と[シュリニヴァーサ・ラマヌジャン](https://ja.wikipedia.org/wiki/%E3%82%B7%E3%83%A5%E3%83%AA%E3%83%8B%E3%83%B4%E3%82%A1%E3%83%BC%E3%82%B5%E3%83%BB%E3%83%A9%E3%83%9E%E3%83%8C%E3%82%B8%E3%83%A3%E3%83%B3 "シュリニヴァーサ・ラマヌジャン")の間の面白い・面白くない数についての議論で、ハーディは彼が乗ったタクシーの番号[1729](https://ja.wikipedia.org/wiki/1729 "1729")を「かなりつまらない」と述べた。しかし、ラマヌジャンはすぐにそれが面白いと答えた。 これは、2通りの「2つの[立方数](https://ja.wikipedia.org/wiki/%E7%AB%8B%E6%96%B9%E6%95%B0 "立方数")の和」で表せる最小の自然数だからである。

このようにすべての数を分類しようとすると、[パラドックス](https://ja.wikipedia.org/wiki/%E3%83%91%E3%83%A9%E3%83%89%E3%83%83%E3%82%AF%E3%82%B9 "パラドックス")または[二律背反](https://ja.wikipedia.org/wiki/%E4%BA%8C%E5%BE%8B%E8%83%8C%E5%8F%8D "二律背反")につながる。[自然数](https://ja.wikipedia.org/wiki/%E8%87%AA%E7%84%B6%E6%95%B0 "自然数")を*面白い・面白くない集合*に[分割](https://ja.wikipedia.org/wiki/%E9%9B%86%E5%90%88%E3%81%AE%E5%88%86%E5%89%B2 "集合の分割")することは失敗している。「面白さ」の定義は通常、主観的で直感的な概念だからである。

「面白い」が客観的に定義されている場合、パラドックスは軽減される。たとえば、[オンライン整数列大辞典](https://ja.wikipedia.org/wiki/%E3%82%AA%E3%83%B3%E3%83%A9%E3%82%A4%E3%83%B3%E6%95%B4%E6%95%B0%E5%88%97%E5%A4%A7%E8%BE%9E%E5%85%B8 "オンライン整数列大辞典")のエントリに現れない最小の自然数は、 [2009年](https://ja.wikipedia.org/wiki/2009%E5%B9%B4 "2009年")6月12日の時点では11630であることが判明していた.[\[1\]](https://ja.wikipedia.org/wiki/%E9%9D%A2%E7%99%BD%E3%81%84%E6%95%B0%E3%81%AE%E3%83%91%E3%83%A9%E3%83%89%E3%83%83%E3%82%AF%E3%82%B9#cite_note-nathanieljohnston-1)。 この定義に適合する数は、2009年11月から少なくとも2011年11月までの間は12407であり、2012年4月の時点で13794になり、 2013年11月から少なくとも2014年4月14日までの間は14228であった[\[1\]](https://ja.wikipedia.org/wiki/%E9%9D%A2%E7%99%BD%E3%81%84%E6%95%B0%E3%81%AE%E3%83%91%E3%83%A9%E3%83%89%E3%83%83%E3%82%AF%E3%82%B9#cite_note-nathanieljohnston-1)。2021年11月現在では、この数は20067となっている。[\[2\]](https://ja.wikipedia.org/wiki/%E9%9D%A2%E7%99%BD%E3%81%84%E6%95%B0%E3%81%AE%E3%83%91%E3%83%A9%E3%83%89%E3%83%83%E3%82%AF%E3%82%B9#cite_note-2)この面白くない数の定義は、OEISが各数列を有限個の要素のみリストアップするため可能である。例えば、 OEIS : [A000027](https://oeis.org/A000027 "oeis:A000027")はすべての自然数の列であり、無限に継続すると、すべての正の整数が含まれる。しかし、数列は77までしかエントリに記録されていない。[\[3\]](https://ja.wikipedia.org/wiki/%E9%9D%A2%E7%99%BD%E3%81%84%E6%95%B0%E3%81%AE%E3%83%91%E3%83%A9%E3%83%89%E3%83%83%E3%82%AF%E3%82%B9#cite_note-3)

数学者で哲学者の[Alex Bellos](https://ja.wikipedia.org/w/index.php?title=Alex_Bellos&action=edit&redlink=1 "「Alex Bellos」 (存在しないページ)")（[英語版](https://en.wikipedia.org/wiki/Alex_Bellos "en:Alex Bellos")）は、2014年に、[247](https://ja.wikipedia.org/wiki/247 "247")が「英語版ウィキペディアに独立した記事を持たない最小の数」であったため、面白くない最小の数の候補になると提案した。 [\[4\]](https://ja.wikipedia.org/wiki/%E9%9D%A2%E7%99%BD%E3%81%84%E6%95%B0%E3%81%AE%E3%83%91%E3%83%A9%E3%83%89%E3%83%83%E3%82%AF%E3%82%B9#cite_note-4)
