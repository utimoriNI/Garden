# オーボエは息が余る

> ほかの管楽器では息が足りんくなって苦しくなっちゃうけど、オーボエは逆に息が余っちゃうねんな。これだけ聞くとなんや楽やなって思うかもしれんけど、息を少しずつ使うっていうことは、つまりは息を止めてるのに近い状態を続けるハメになるってことなわけ。演奏しているうちに肺のなかに二酸化炭素がたまっちゃって苦しいん
