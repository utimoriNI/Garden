# 既読なんて以ての外だ

> 既読なんて以ての外だ。あれはもう爆弾の導火線に火を点けるようなもので、既読をつけたその瞬間から、最適な返信を生み出さなきゃ死ぬ、みたいな地獄のレースが勝手に始まる。その状況を予想しただけで、脳が疲れる。だから先延ばしにしてしまう。
