# 「地元に残った子達が結婚が早い理由も分かったよ

> 「地元に残った子達が結婚が早い理由も分かったよ。だって、迷う必要がないもん。子供を産んでも自分の親に頼れるし。結婚相手が同級生だったりしたら尚更だよね。お互いの親のことも知ってるし、どちらの実家も近い。助けてくれる人の数が違う。勿論、頼れない人もいるだろうけど、地元から出た人と比べたら物理的な障壁の数が違う」
