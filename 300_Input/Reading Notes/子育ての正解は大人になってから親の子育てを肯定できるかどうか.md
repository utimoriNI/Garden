# 子育ての正解は大人になってから親の子育てを肯定できるかどうか

> 「成長した子どもが、大人になってから親の子育てを肯定できるかどうか」 　スミちゃんが言って、私を見た。 「人生は長いからさ。大人になってから子どもに自分がやってきたことを肯定してもらえないと、いざ対等な状態になった子どもに見捨てられることになるよ。感謝されないし、仲良くしてもらえない。保護者と被保護者はいずれ、介護だなんだで逆転するんだしさ」
