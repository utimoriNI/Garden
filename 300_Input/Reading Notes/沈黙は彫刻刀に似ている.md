# 沈黙は彫刻刀に似ている

> 沈黙は彫刻刀に似ている。冷蔵庫の音や隣の部屋の生活音、外の世界を人や車が通り過ぎていく音──それまでもずっとそこにあったはずの音たちを、空間からはっきりと削り出す。その場にあったのに感知できていなかっただけの何かを、突然表出させてしまう。
