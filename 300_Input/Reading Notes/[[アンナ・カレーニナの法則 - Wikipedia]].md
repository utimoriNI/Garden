# [[アンナ・カレーニナの法則 - Wikipedia]]

> トルストイの指摘はひとつの原則であり、男と女の結婚生活以外にも、いろいろな事柄にあてはまる。われわれは、成功や失敗の原因をひとつにしぼる単純明快な説明を好む傾向にあるが、物事はたいていの場合、失敗の原因となりうるいくつもの要素を回避できてはじめて成功する。トルストイの「アンナ・カレーニナの原則」は、まさにこの点をいいあてている。そして人類史を大きく変えた動物の家畜化の問題も、この原則によって説明
