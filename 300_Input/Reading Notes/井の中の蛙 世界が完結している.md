# 井の中の蛙 世界が完結している

> この人たちは──世界が完結しているのだ。 　自分の目に見える範囲にある情報がすべてで、その情報同士をつなぎあわせることには一生懸命だけど、そこの外に別の価値観や世界があることには気づかないし、興味もない。 　──井の中の 蛙、という言葉を思い出した。とても狭い範囲の常識と知識で生きている。
