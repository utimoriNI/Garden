# 体内の宗教

> 体内の宗教が同じ人の死は、当人の死のみに収束しない。その死は、同じ宗教の他者を殺すことでもある。 翻って、宗教が同じ人が心身共に健康で生きているというだけで、手放しそうになる明日を手繰り寄せられるときがある。その人が生きている世界なら自分も生きていけるのかもしれないと、そう信じられる瞬間が確かにある。
