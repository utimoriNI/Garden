# 人は簡単に測定できる要素に注目することで問題を単純化する

> 人間には、もっとも簡単に測定できる要素に焦点を絞ることで問題を単純化するという性質がある（１）。だがもっとも簡単に測定できるものがもっとも重要なものであることはまれで、実際にはまったく重要ではない場合がある。これが、測定基準の欠陥の中で第一の要因だ。

> 求められる成果が複雑なものなのに、簡単なものしか測定しない というものだ。ほとんどの仕事には複数の責任が伴い、ほとんどの組織には複数の目標がある。たったひとつの責任または目標に測定を集中させることは、しばしば欺瞞的な結果につながる。
