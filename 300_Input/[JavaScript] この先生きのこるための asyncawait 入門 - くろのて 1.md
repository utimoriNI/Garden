# 「結果」はどう生まれるのか｜イラレ職人月報2023/12



実力はあるのに結果が出ない。対した実力はないのに人気がある。

古今東西、リアルとSNSを問わず言われ続けてきた言葉です。理不尽と感じる人はたくさんいるでしょうし、だからこの世は面白いとも言えます。

なぜこんな状況になるのか。  
あくまで視点の一つですが、**「デザイン」や「イラスト」といった技術や要素を、１つの円として考える**とわかりやすいと思います。

「デザイン」や「イラスト」などのスキルをグレーの円で考える

## 「結果」はどう生まれるのか

つい自分たちの領域だけで良し悪しを考えがちですが、前提として、デザインやイラストという**単独の技術だけで「結果」は生まれません**。

### 「人気クリエイター」に必要なのは

SNSで活動するイラストレーターで考えてみましょう。極端な話、どれほど素晴らしいイラストを描けても、フォロワーが0人では拡散はしません。「拡散力」という**別の円を重ねることで、色はより濃くなり目立つようになります。**
この「濃い範囲」の目立ち具合が、結果につながると考えてください。

[![画像](https://assets.st-note.com/img/1704026916126-ZkxojxA1JX.png?width=800)](https://assets.st-note.com/img/1704026916126-ZkxojxA1JX.png?width=2000&height=2000&fit=bounds&quality=85)

イラスト単独より、他の円と重ねるほうが色は濃い

もちろん「イラスト」の能力をひたすらに高めて「色を濃くする」ことはできます。「実力で評価される人」はこの状態です。しかし他の人と比べても突出して目立つほど「濃くなる」のが至難の業なのは言うまでもありません。

一方で、他の円と重ねることで色は簡単に濃くできます。スキルの掛け合わせってやつです。「拡散力」だけではなく、「人脈」とか「映像」とか「専門知識」とか、身も蓋もないですが「容姿が良い」も立派なスキルの１つです。

[![画像](https://assets.st-note.com/img/1704028819719-DQdKtEzQYo.png?width=800)](https://assets.st-note.com/img/1704028819719-DQdKtEzQYo.png?width=2000&height=2000&fit=bounds&quality=85)

右の人の方が色は濃くなる

つまり **「結果」とは「総合力」から生まれる** ものであり、RPGで例えるなら「イラスト」は「攻撃力」といったパラメータの１つに過ぎません。ラスボスをワンパンで倒せるならともかく、大多数は防御力や素早さも必要です。クリエイティブの技術だけで結果を考えるのは、目隠しをして銃を撃つようなものだと僕は考えます。

  

### クライアントワークの場合

ここまではSNSで考えましたが、「仕事」となれば話はもっと膨らみます。

ロゴやポスター、Web、SNSといった様々な媒体はもちろん、経営者、営業、広報、事務員さん、クライアントから販売員まで、ありとあらゆる要素で「仕事」は成り立っています。

[![画像](https://assets.st-note.com/img/1704025326190-QbLwTnsNKT.png?width=800)](https://assets.st-note.com/img/1704025326190-QbLwTnsNKT.png?width=2000&height=2000&fit=bounds&quality=85)

あらゆる要素で「仕事」は成り立つ

そしてこれらの円も、重ねなければ薄いままです。例え高い技術があっても、見当違いの場所を塗っていては「良い結果」は生まれません。

[![画像](https://assets.st-note.com/img/1704029200744-hTfNRQqVhL.png?width=800)](https://assets.st-note.com/img/1704029200744-hTfNRQqVhL.png?width=2000&height=2000&fit=bounds&quality=85)

「デザイン」だけ別の場所を塗っている状態

「デザイナーはコミュ力が重要」とか「デザイナーもビジネスを学ぶべき」といった意見もこれに近いと思います。全体を見て足並みを揃えたり、的確な判断ができるデザイナーは良い結果を生み出しやすくなります。

また「デザイナーはデザインだけに集中すべき」と言う意見も、一面では真理だと思います。ただしそれは的確な判断ができるディレクターや、経験豊富なオペレーター、その他の業務の人員がバッチリ揃っている環境に限ります。「必要がない」のではなく、「誰かがやってくれている」わけです。

じゃあ結局どうすれば良いかと言えば、ケースバイケースに過ぎないので「こうすればOK」と言い切るのは乱暴です。ここまでの理屈の上で、自分の環境の場合はどうすればいいか、自分で判断をしましょう。

  

## 「濃い」結果を生むには

より濃い場所を作る上で重要になるのは、**「重なり」の的確さ**です。同じ濃さと数の円でも、重なり方によって結果は変わります。社長やプロデューサー、ディレクターの采配によって結果が変わるのはこういうことですね。

[![画像](https://assets.st-note.com/img/1704029654957-Ge0JBrEupY.png?width=800)](https://assets.st-note.com/img/1704029654957-Ge0JBrEupY.png?width=2000&height=2000&fit=bounds&quality=85)

また、人手の多い大企業の方が円の大きさや数で有利ですが、人数が増えるほど統制を取るのは難しくなります。的確な経営ができていない企業より、個人のインフルエンサーの方が濃いエリアを作りやすい、なんてこともあるでしょう。

[![画像](https://assets.st-note.com/img/1704029788033-aQB3HInY7L.png?width=800)](https://assets.st-note.com/img/1704029788033-aQB3HInY7L.png?width=2000&height=2000&fit=bounds&quality=85)

円が小さいインフルエンサーの方が、特定のエリアが濃い場合

要するに、明確な「目的」に対して的確なスキルを采配できるかどうかです。ビジネスで「経営理念」や「ビジョン」といったものがあるのは、別に形式やカッコいいからではなく、こうしてスキルや人員の「統制」に（的確に運用できていれば）有効だからだと僕は思います。なんでも闇雲にやれば良いわけではないのですね。

  

## 「外側」にも要素はある

ここまでの説明は、個人や企業の「内側」の要素だけです。実際には「外側」に競合企業があり、トレンドや時世があります。

例えば競合する相手が同じところに集まれば、色が濃くても目立ちにくい。  
（逆に密集してるからこそ目立つ場合もありそうですが）

[![画像](https://assets.st-note.com/img/1704032194285-PAkWuyzzU8.png?width=800)](https://assets.st-note.com/img/1704032194285-PAkWuyzzU8.png?width=2000&height=2000&fit=bounds&quality=85)

また、話題性の高い場所や人（消費者）の集まっている場所にいる方が、同じ円でも色は濃くなります。背景色はきっとグラデーションのように複雑で、常に変化し続ける流動的なものです。

[![画像](https://assets.st-note.com/img/1704032360371-E4KzYi5lQ9.png?width=800)](https://assets.st-note.com/img/1704032360371-E4KzYi5lQ9.png?width=2000&height=2000&fit=bounds&quality=85)

背景にも色がある

つまり「結果」とは、内側だけではなく外側の様々な要素も合わせることでようやく決まるのです。「実力があるのに結果が出ない」のは、実力以外の部分で結果を出す状況が整っていない、となります。

労力が報われるとは限らない無常な仕組みですが、逆に言えば多くの人に逆転の目があるやりがいのあるゲームだと僕は思います。来年はどんな色になり、どこに自分の色を置こうか、今からワクワクします。

それでは良いお年を。
