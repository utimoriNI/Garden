# 雑談力
###### メモ
- p24 つかみが大事
	- その話を聞いてみたいと思わせる事が大切
	- バーバリライオンの話
		- トラとライオンが戦ったらトラが勝つと思ったって話から入るの天才
		- 虎の方がでかかった→なんで百獣の王？→バーバリライオンってやつがいた
			- この構成だとただのうんちくひけらかしって感じしない
- p39 単純なうんちくほどつまらないものは無い
	- 一言で片付くうんちくは無意味。エピソードを加えて膨らませよう
	- 長野県岩松院に葛飾北斎の作品があるらしい
	- 情報というのは白黒の線で描いた絵
		- 色をつけて背景を描き加えると絵が動き出す
		- 意外な事実ではなく、人間性が生み出すストーリーに惹かれる
- p52 面白い話にはストーリーがある
	- 箇条書き的な知識は面白くない
	- 物語になってる話を聞きたい
	- 教科書がつまらないのはそこに物語がないから
	- ※如何に短くストーリーテリングを行う？
- p62 面白い話にするために
	- 難しそうな話にストーリーを付ける
	- 微積分が生まれた理由。四色定理は印刷業者は経験から学んでいた
	- ※ただの一人語りになっていいのかな？以下に他人に話を振れるか
- p69 ストックを持とう
	- 面白い話ができるようになりたければ面白い話をストックする
	- コンテンツに触れる時、面白い話と思えばそれを覚えるという気持ちを持つ
- p96 相手ではなく、自分が関心を持つ話題を探せ
	- 自分が関心を持ったことを話す、話のテーマを聞き手は何も知らないという前提で話す
- p108 スターになりたいでも構わないが、大前提は人を楽しませたい
- p113 面白さの7割以上が話術
- p118 話が上手くなる1番の方法は、経験を積むこと
	- 人が面白がったところをさらに膨らませ、退屈したところを短くしていく
- p127 とっておきの練習法 映画や小説の話をする
	- その作品が何を描いた物語なのかを、端的に表現する
-  p164 個人的な思い出話でも普遍性を持たせればok
	- 幼い頃の怖い記憶などの普遍的なテーマに沿った話であれば、他の人も話を続けやすい
- 総括
	- 話の構成という抽象的な話は面白かった
	- ストーリーを楽しむのはまさにそうだなと思った
		- 取っ付き難い話を楽しく聞けたり、解像度を高く理解できる気がする
		- 人の性格とか趣味とかもストーリーで聞いてみると面白いかも
