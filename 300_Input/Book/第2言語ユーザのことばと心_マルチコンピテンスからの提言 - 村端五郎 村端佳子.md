#### マルチコンピテンスの3つの考え方
- 1. 第二言語ユーザの言語知識・能力を記述する際に、ネイティブスピーカーの言語知識・能力を基準として比較して、「正しい」「逸脱している」と考えるのは無意味
- 2. 2つの言語が第二言語ユーザの頭の中で相互に作用しているなら、母語→第二言語の影響と第二言語→母語の相互作用があるはず
- 3. 第二言語学習は、第二言語ユーザの言葉だけでなく、思考・判断・知覚などの認知活動に影響を与える

#### 第二言語が母語の語彙に影響を与える例
- 日本語や英語のモノリンガルとそれらのバイリンガルに「ボス」「スマート」「タレント」「バイク」という単語から思いつくものを書いてもらう

|                          | ボス           | スマート       |
| ------------------------ | -------------- | -------------- |
| 英国在住の日本語母語話者 | 会社、仕事     | 利口、おしゃれ |
| 日本在住の日本語母語話者 | サル、リーダー | ほっそりした、痩せた               |

- 人の相槌の研究
	- 外国人視点だと日本語話者は相槌が多く、話を中断されたりどの意見に同意しているかがわからない
- 日本人と中国人が誘いを断るときに使う言葉の違い
	- 中国人は「行けない」「考えておく」という、相手に呼びかけるための言葉を使う
	- 日本人は「行きたいのですが」のようなためらいの言葉を多く使う
