# 学力喪失　認知科学による回復への道筋
### 1章
- **問題提起**
	- 小学生は計算はできても算数の文章問題を解けない
		- 子供が14人、1列に並んでいます。ことねさんの前に7人います。ことねさんの後ろには、何人いますか。
			- 14-7で7人と答える人が多い
	- 単位の変換が身体の一部になっていない
		- 計算の中で1時間を60分に変換できるときもあれば、100分と変換してしまう時もある
		- 「あることをちょっと知っている」「表面的に知っている」ということは、その知識が、必要な時にいつも一貫して使える「生きた知識」になっているとは限らない
	- 算数の文章題をきちんと解くことができるようになるためには、単位の変換が身体化され、意識を向けなくても自動的に必要な形で使えることが必要
- **1章まとめ**
	- 全ては意味の不理解に落とし込まれる
		- 数字や演算記号のそれぞれの意味、方程式の意味も腑に落ちていない
		- 小学生の時に理解できなかった分数・割合の基本概念が理解できていない状態が続いている
		- 計算ができたとしてもその意味が分かっていない

### 2章
- **知識に対する誤った認識**
	- 知識は以下のように誤解されている
		- 知識は「客観的な事実」の集積
		- 知識は多ければ多い程良い（モアイズベター信仰）
		- 知識は分かりやすく教えれば相手の脳に移植できる
	- 算数文章題を例に挙げれば、小学生は「問題文に出てくる順番に式に数字を入れる」という方略を持っていることで間違ってしまう
		- それに対して「問題文に出てくる順番に式に数字を入れてはいけない」と指導しても効果はない
			- **フレーム問題だ！**
	- 知識とは「学び手が創り上げていくもの」であり「生きた知識」と「死んだ知識」がある
		- 生きた知識と死んだ知識は2分できるものではなく、グラデーションである
		- 生きた知識とは「様々な状況においてすぐに取り出せて問題解決に使える知識」といえる
		- 例：プロ棋士は将棋の盤面を一瞬で覚えられる
- **スキーマなしでは学習できない**
	- スキーマとは学習者が経験から導出した暗黙の知識
	- 認知科学では、学習のつまずきのもっとも大きな原因が、スキーマの誤りであると考えられている
	- 母語は究極の生きた知識だが、大人に丁寧に教えられて習得するわけではない
		- 母語の仕組みや単語の意味を自分で考え、手探りで試行錯誤を繰り返しながら身に着ける
