# 他者といる技法
- 原型としての社会
	- 人には他者からの承認が必要
		- アイデンティティは他者からの承認が必要な不安定なもの
		- 他者も主体でありコントロール不可能
	- 他者を自分を承認する客体とみなすか自身と同じ主体とみなすかでそれぞれの存在証明権を奪い合う
