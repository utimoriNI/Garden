# 「推し」の科学プロジェクション・サイエンスとは何か
### メモ
- p18 来談者中心療法
	- 推しに救われたという経験は、推しによって自分を取り巻く世界の捉え方が変わった
- p19 推しの定義
	- その対象をただ受け身的に愛好するだけでは飽き足らず、能動的に何か行動してしまう対象が推し
	- 
- p21 認知科学の身体性認知
	- 身体や環境や感情は、人間の認知活動と結びついている
	- 応援すると魅力を感じる
		- ペンライトを振る実験
- p25 ミラーニューロンと報酬系
	- 他者の運動が自分の運動のように感じる
	- 他者の報酬が自分の快感になる代理報酬でも報酬系が活性化する
	- 応援する→他者の行動を自分の行動のように捉える→対象の成功が代理報酬をもたらす
- p33 認知科学のプロジェクション
	- 自分がどのように世界を認識するかだけでなく、自分は認識をどのように世界へ付加していくのか、心と世界はどのように繋がっているのかを研究するのがプロジェクション
	- 人間は物理世界からの情報を受け取り、処理をして、表象(認識)を作り出す
	- そして、作り出した表象を物理世界に反映することで世界に意味付けをしてその中で活動する
- p42 投射の種類
