# MBTIへの招待
###### メモ
p4,5
- ユングのタイプ論は、私達の意識の向け方、気づきに対する秩序のもたらし方、決定の下し方を検討するもの
- タイプ論は行動予測には使えない
	- 人の行動は様々なものから影響を受けて決まる
	- ○○タイプは、○○タイプに特徴的な傾向と捉える方が良い
	- タイプだけでその人の行動を全て説明できない
- p6 心理学的タイプ論のモデル概要
	- ユングは心理学的類型1,2で人間の心の習慣を3指標で分類できるとしている
		- 1. 心的エネルギーの源と方向(**内向Introversion・外向Extraversion**)
		- 2. 情報をどのように知覚し、どのような種類の情報に惹き付けられるか(**感覚機能Sensing・直感機能Intuition**)
			- 感覚機能：五感を通じて認識される知覚データを収集する
			- 直観機能：認知した物事の関連性や抽象的なデータをひらめきを通じて収集する
		- 3. 収集した情報から結論を導いたり判断する機能(**思考機能Thinking・感情機能Feeling**)
			- 思考機能：物事について因果関係の分析に基づいて判断する
			- 感情機能：物事を個人的な価値観や対人関係への影響に基づいて結論を導く
	- キャサリン・ブリッグスとイザベル・ブリッグス・マイヤーズはさらに4つ目の指標を加えた
		- 外の世界に対する接し方についての心の習慣を確認するもの(**判断的態度Judging・知覚的態度Perceiving**)
			- 判断的態度：外界を体系立てたりまずその状況を判断しようとする
			- 知覚的態度：外界から新しい情報を取り入れてその場で臨機応変に対応しようとする
- p12 外向と内向
	- 外向・内向は心の目の向けられている実際の方向
	- 外向の場合は外界にエネルギーの源を求めるためにエネルギーを放ちながら、外界からエネルギーを補給している
	- 内向の場合は内なる世界から湧き出る思いや考えにエネルギー源を求めるために内界からエネルギーを補給している
- p25 判断的態度(J)と知覚的態度(P)
	- 外界に対しての振る舞い方
	- また、外界に対して使っている機能の反対が、内側に対して使われるとしている
	- J：物事を決めようとしたり、体系立てようとしたり、順を追って話をする傾向
		- 物事に優先順位をつけ、綿密な計画のもと動こうとする
		- 反面、突然の変更に対してもっと上手く計画すれば良かったと考えたり、自分が望む結果に惹かれすぎて、もっと良い結果をもたらすきっかけを逃すことがある
	- P：その場その場に対応しようとしたり、枠を持たずに臨機応変にしようとしたりする
		- 具体的な計画に従うことは出来るが、後になって良い情報を得られたらどうするだろう？と迷う
- p39 タイプダイナミクスの基本的な目的
	- 2つの相反する極はタイプ構成の基底
	- 主機能と補助機能はバランスよく使うことが健全な精神のために必要
		- INTJの場合は、内向に働く直感だけでなく、外交に対する思考機能を時々使う必要がある
