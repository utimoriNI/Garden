# 入門!論理学
### メモ
- p10 論理的であることは常識的であることと違う
	- 真面目であることとお金持ちであることが違うように、2つに関係がない
	- 対偶をとると分かる
- p13 推測と推論
	- 常識や経験から判断した結論の導き方は推測
	- 論理的な結論の導き方は推論
- p19 論証と導出
	- ある前提から結論を導くこと全体を論証
	- 前提から結論を導く過程を導出
- p23 いい問題
- p38 Aを否定するということはAという主張が間違いであることを訴えることで世界の状態を描写する
- p50~
	- 排中律を認める論理体系と認めない論理体系
	- 排中律を認める論理体系では曖昧な概念を扱わず、有るか無いかのどちらかと言える概念を扱う
	- 実在論的
- p58~ 二重否定則
	- 二重否定則は2つに分けられる
		- 1. Aならば(Aでは無い)ではない
		- 2. (Aではない)ではないならばA
	- 排中律を認めた論理体系ではどちらも主張できる
	- 排中律を認めない体系では2は言えない
- p68 矛盾律
	- AかつAでない←これを認めない
	- 排中律を認めない人でも矛盾律は認める
		- 富士山か曖昧でも｢富士山であり、かつ富士山ではない｣とは違う
		- ｢結婚すると言った覚えは無いけど、結婚しないと言ったこともない｣と「結婚しよう、結婚しない」ということは違う
- p78 基本的の定義
	- 否定では、排中律や二重否定取りを論理法則として認めるかどうかが、否定の意味の捉え方に繋がり、それが異なる論理体系に繋がった
	- 否定や「かつ」などの論理法則は、論理学の中で、どのように規定されるかによってどの体系を扱っているかが変わる
- p123 AならばBとAならば(Bではない)は両立する
	- Aを成立すると仮定すると矛盾が生じる→Aという仮定が間違っている
	- 直感的に変だと思うのは、Aが成立する場合があるという前提に立っているから
- p168 論理法則として採用されているものは反例が見いだせないもの→
