Raindrop.ioのブックマークを自動同期し、本文抽出・要約・embedding・ハイブリッド検索・MCP連携・管理UIまで備えた、自分用のセマンティック検索ツールを完成系として実装してください。

目的:
Raindrop.io Proのセマンティック検索に近い体験を、自前のローカル/個人用ツールとして実現したいです。通常検索UI、LLMから使うMCP、同期・再index管理UIをすべて含めてください。

必須機能:
- Raindrop.io APIからブックマーク、tags、collections、notes、highlights、excerpt、更新日時を同期
- 差分同期
- URL先の本文抽出
- HTML本文抽出
- PDFは可能なら対応、難しければ拡張しやすい設計
	- Youtubeなどの動画の場合は字幕の情報などから内容を抽出する
- title / excerpt / note / tags / highlights / 本文チャンク / 要約を保存
- LLMで要約、自動タグ、キーフレーズ生成
- embedding生成
- ベクトル検索
- キーワード検索
- ベクトル検索 + キーワード検索のハイブリッド検索
- LLMによる検索結果の再ランキング、または再ランキング可能な設計
- Web UIで検索、フィルタ、結果詳細、元URL表示
- 管理UIで同期状況、失敗URL、再同期、再embedding、抽出本文確認
- MCP ServerでLLMクライアントから検索できる
- CLIも可能なら追加
- .env.exampleを用意し、秘密情報は直接書かない
- READMEにセットアップ、同期、検索、MCP設定方法を書く
- 最低限のテスト、または動作確認コマンドを用意
- 

推奨構成:
- 検索API
- Web UI
- MCP Server
- 同期/Indexer Worker
- Metadata DB
- Text Store
- Vector DB

完成条件:
- READMEの手順通りに環境変数を設定すれば、Raindrop.ioから同期できる

- MCPから同じ検索を呼べる
- 管理画面で同期状態と失敗理由が確認できる
- 主要な処理にエラーハンドリングとログがある
- 実装後に起動確認と主要フローの検証を行い、結果を報告する
