## 単語管理の方法について
### 要件
| 属性名   | 意味               | 型             |
| -------- | ------------------ | -------------- |
| 単語     | 気になった単語     | 文字列         |
| 意味     | 単語の意味         | 文字列         |
| カテゴリ | 見つけた状況、特徴 | マルチセレクト |
| コメント | 気になった理由     | 文字列               |

- カテゴリごとにフィルターできると嬉しい
- どのデバイスからでも追加できるようにしたい
- できればObsidianで管理したい
