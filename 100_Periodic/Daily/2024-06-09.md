画像内の定積分の式について

画像内の定積分の式は、1か月の水道使用料金の期待値を求めるものです。

式は以下の通りです。

$E[g(x)] = \int_{-\infty}^{\infty} g(x) f(x) dx$

ここで、

- E[g(x)]: 1か月の水道使用料金の期待値
- g(x): 水道使用量 x における水道料金
- f(x): 水道使用量 x の確率密度関数

水道料金 g(x) は、以下の式で定義されています。

g(x) = \begin{cases}

1000, & 0 \leq x < 10 \\

1120, & 10 \leq x < 15 \\

1280, & x \geq 15

\end{cases}

水道使用量 x の確率密度関数 f(x) は、一様分布であると仮定されています。

この式を解くために、まず、水道使用料金 g(x) を3つの部分に分けて積分します。

$E[g(x)] = \int_{0}^{10} g(x) f(x) dx + \int_{10}^{15} g(x) f(x) dx + \int_{15}^{20} g(x) f(x) dx$

次に、各部分積分において、水道料金 g(x) を定数と見なし、積分を行います。

$E[g(x)] = \int_{0}^{10} 1000 f(x) dx + \int_{10}^{15} 1120 f(x) dx + \int_{15}^{20} 1280 f(x) dx$

水道使用量 x の確率密度関数 f(x) は一様分布であると仮定しているので、積分結果は以下のようになります。

$E[g(x)] = 1000 \int_{0}^{10} f(x) dx + 1120 \int_{10}^{15} f(x) dx + 1280 \int_{15}^{20} f(x) dx$

ここで、一様分布の確率密度関数は以下の式で定義されています。

f(x) = \frac{1}{b - a}, \quad a \leq x \leq b

この式を代入すると、以下のようになります。

$E[g(x)] = 1000 \cdot \frac{1}{10 - 0} + 1120 \cdot \frac{1}{15 - 10} + 1280 \cdot \frac{1}{20 - 15}$

計算すると、以下のようになります。

$E[g(x)] = 1000 + 112 + 128 = 1240$

よって、1か月の水道使用料金の期待値は1240円となります。

式の解説

式

$E[g(x)] = 128 \int_{0}^{20} (1 - \frac{x}{20}) dx + (112 - 128) \int_{0}^{15} (1 - \frac{x}{20}) dx + (100 - 112) \int_{0}^{10} (1 - \frac{x}{20}) dx$

は、以下の式を変形したものです。

$E[g(x)] = \int_{0}^{10} 1000 f(x) dx + \int_{10}^{15} 1120 f(x) dx + \int_{15}^{20} 1280 f(x) dx$

この式は、水道料金 g(x) を3つの部分に分けて積分したものです。

各部分積分において、水道料金 g(x) を定数と見なし、積分を行います。

$E[g(x)] = 1000 \int_{0}^{10} f(x) dx + 1120 \int_{10}^{15} f(x) dx + 1280 \int_{15}^{20} f(x) dx$
